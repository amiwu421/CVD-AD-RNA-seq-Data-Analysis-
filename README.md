# RNA-seq Data Analysis Pipeline - Cardiovascular Disease Stage Specific Analysis

## Overview
This project contains an R-based workflow for analyzing transcriptomic data. 
The analysis includes data preprocessing, differential expression analysis, pathway enrichment analysis, overlap analysis, and visualization.

## Objectives
- Identify differentially expressed genes
- Perform functional pathway analysis
- Visualize biological patterns from transcriptomic data

## Tools & Packages
- R
- R Markdown
- ggplot2
- DESeq2
- clusterProfiler

## Files
- `analysis.Rmd` - Complete R analysis workflow
- `analysis.html` - Rendered analysis report

## Results
The analysis generates visualizations and enrichment results to investigate biological pathways associated with the dataset.
