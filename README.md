# CVD-AD RNA-seq Data Analysis

## Overview

This project contains an R-based bioinformatics workflow for analyzing RNA-seq transcriptomic data to investigate gene expression patterns associated with cardiovascular disease (CVD) and Alzheimer's disease (AD).

The analysis focuses on processing transcriptomic datasets, performing differential expression analysis (DEA), and identifying biological pathways associated with disease-related phenotypes.

## Objectives

- Perform RNA-seq data preprocessing and quality filtering
- Identify differentially expressed genes between experimental groups
- Analyze gene expression patterns associated with CVD and AD
- Generate visualizations and downstream biological interpretations

## Data

The project uses transcriptomic data from the ROSMAP cohort, including normalized gene expression data.

Due to file size limitations and data management considerations, raw datasets are not included in this repository.

## Methods & Tools

- R
- R Markdown
- Differential Expression Analysis (DEA)
- Gene expression analysis
- Data visualization

Main analysis workflow:
- Data loading and preprocessing
- Filtering and normalization
- Differential expression analysis
- Visualization of results
- Biological interpretation of gene expression changes
